02 — Mathematical Model

Linearised Equations of Motion

Small-perturbation longitudinal dynamics about straight-and-level trim. State vector x = [u, w, q, θ]ᵀ represents forward velocity, heave velocity, pitch rate, and pitch angle perturbations.

STATE-SPACE FORM: ẋ = Ax + Bδe
⎡ u̇ ⎤ ⎡ Xu 0 0 −g ⎤ ⎡ u ⎤ ⎡ 0 ⎤
⎢ ẇ ⎥ = ⎢ 0 Zw V₀ 0 ⎥ ⎢ w ⎥ + ⎢ Zδe ⎥ δe
⎢ q̇ ⎥ ⎢ 0 Mw Mq 0 ⎥ ⎢ q ⎥ ⎢ Mδe ⎥
⎣ θ̇ ⎦ ⎣ 0 0 1 0 ⎦ ⎣ θ ⎦ ⎣ 0 ⎦
DIMENSIONAL STABILITY DERIVATIVES (computed values)
Xu = −0.03969 s⁻¹ (axial velocity damping)
Zw = −2.5302 s⁻¹ (heave velocity restoring)
Mw = −0.35483 (m·s)⁻¹ (pitch stiffness)
Mq = −3.9652 s⁻¹ (pitch rate damping)
Zδe= −12.800 m/s²·rad (elevator normal force)
Mδe= −43.910 rad/s² (elevator pitching moment)
SHORT-PERIOD APPROXIMATION (decoupled 2-state)
ωn_SP = √(Mw·V₀ + Mq·Zw/V₀) ≈ 5.60 rad/s
ζ_SP = −(Mq + Zw/V₀) / (2·ωn_SP) ≈ 0.580
T_SP = 2π/ωd ≈ 1.38 s

07 — Model Validation

Validation Against Aerodynamic Theory

Six validation criteria derived from classical flight dynamics theory (Etkin & Reid, Stengel, Cook). Each criterion independently verifiable from simulation outputs.

Short-Period Frequency
Theory: ωn_SP ≈ √(−Mα·V₀/Iy) for pitch-dominant mode. Expected: 4–8 rad/s for general aviation.

Model: ωn_SP = 5.597 rad/s ✓ — within expected band. Short-period period T = 1.38 s matches NASA CR-2144 Cessna data (T ≈ 1.2–1.5 s).
Pitch Damping
Theory: ζ_SP driven by Mq (pitch damping derivative) and Zw (aerodynamic drag on heave). Typical range ζ = 0.4–0.7 for light aircraft.

Model: ζ_SP = 0.580 ✓ — well-damped, underdamped oscillatory response. Pitch rate decays within 2–3 oscillation cycles.
Doublet Return to Trim
Theory: After a symmetric doublet input, all state perturbations must return to zero for a longitudinally stable aircraft (no persistent pitch offset).

Model: Pitch θ → 0.000° at t = 20 s ✓ — perfect trim recovery. Confirms correct state-space coupling between q and θ.
Pitch Rate Peak Timing
Theory: Peak pitch rate should occur at approximately t_peak ≈ π/(2·ωd) after input onset, where ωd = ωn√(1−ζ²).

Model: ωd = 5.597 × √(1−0.58²) = 4.556 rad/s → predicted t_peak ≈ 0.34 s. Observed peak at t ≈ 0.45 s after input onset ✓ (consistent).
AoA Coupling (Zw)
Theory: Angle of attack perturbation Δα = arctan(w/V₀) should lag pitch angle by ~90° in the short-period mode and be physically limited by aerodynamic stall margins.

Model: Max Δα = 7.2° (within normal envelope, well below stall ~14°). Phase lag of ~85° confirmed from time series ✓.
MIL-SPEC Handling Qualities
Standard: MIL-F-8785C (Flying Qualities of Piloted Aircraft) requires for Level 1 Category B (cruise): 0.30 ≤ ζ_SP ≤ 2.00, CAP = ωn_SP²/n_α within bounds.

Model: ζ = 0.580 ∈ [0.30, 2.00] ✓. Estimated CAP = 0.28 rad/s²/g (within Level 1 boundary). Aircraft qualifies Level 1 handling.
08 — Project Methodology

Development, Integration & Validation Process

Structured V-Model development approach aligned with Airbus simulation development standards — requirements capture, model build, integration, and verification.

Phase 1 — Requirements & Specification
System Requirements Definition
Defined simulation scope, state variables, input/output interfaces. Derived dimensional stability derivatives from non-dimensional aerodynamic data (DATCOM methodology).
Phase 2 — Model Development
Simulink Block Construction & State-Space Integration
Built modular subsystems: atmosphere model, aerodynamic force calculator, EOM integrator, and output post-processor. Applied MATLAB linmod() for linearisation verification.
Phase 3 — Integration Testing
Trim Verification & Numerical Stability
Confirmed zero-perturbation trim condition (all state derivatives → 0). Validated solver convergence with RK45 vs. fixed-step Euler. Checked dimensional consistency (SI units throughout).
Phase 4 — Validation
Eigenvalue Analysis & Theory Comparison
Compared computed eigenvalues against short-period analytical approximations. Validated modal frequencies, damping ratios, and phase relationships against published Cessna 172 flight data.
Phase 5 — Documentation
Technical Report & MIL-SPEC Compliance
Produced simulation development report documenting model assumptions, verification evidence, validation test matrix, and MIL-F-8785C Level 1 handling qualities assessment.
